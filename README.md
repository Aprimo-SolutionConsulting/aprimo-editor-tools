# Aprimo Editor Tools 

A sample app to connect to Aprimo using PKCE auth.  

> **This is a community-supported project and is not officially maintained or supported by Aprimo.**

## Available Capabilities

Framework
- An endpoint to receive pagehook calls from Aprimo and send them to a basket page or a single item page

The following tools are available/in development:

- my-basket 
- my-item
- excel-import (in development)

### my-basket

Render the content of the basket from aprimo as a grid or excel based contact sheet.

### my-item

An example of how to send the record id of a single item to the application
